# Cài đặt npm
```js
npm i react-router-dom
```

# `React` giúp render dữ liệu nhưng lại không tự có `router`
## Nó giữ cho giao diện của ứng dụng đồng bộ với URL trên trình duyệt.
## Cho phép bạn định tuyến luồng dữ liệu.
## Giống như bạn nói rằng, với URL này thì sẽ có router và giao diện tương ứng.
